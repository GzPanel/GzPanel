GzPanel

The panel for all your hosting needs!

The functionality of this panel is to support as many applications possible, yet make it easy to add new application support. In the initial stages, the panel will have strong support for Linux systems.

This panel is also currently open source, therefore if you are able to lend a helping hand, please do so.

More info will be added at later stages.
